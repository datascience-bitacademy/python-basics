2. virtualenv   : python2부터 사용했던 가상환경 라이브러리       o

4. conda          : Anaconda Python 설치후, 사용가능하다.

 venv로 파이썬 프로젝트 가상환경 만들기

1. 프로젝트 디렉토리 만들기
# mkdir log-analysis

2. 프로젝트 디렉토리로 가기
# cd log-analysis

3. 가상환경 생성하기
# python3 -m venv venv

4. 가상환경 구동(activate)
# source venv/bin/activate

5. 확인
(venv) # python
